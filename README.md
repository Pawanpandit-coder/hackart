# MERN Project for hackers lovers simple with various undergoing hacking new tools and technologies.

## Tech Stack
- MongoDB
- Express
- React
- Node.js

## Setup

### Backend
cd server  
npm install  
Create `.env` file and add `MONGO_URI`  

### Frontend
cd client  
npm install  

### Run Project
# Backend
cd server
node main.js

# Frontend
cd client
npm run dev

## Setup
1. Clone repo
2. Run `npm install`
3. Create `.env` file
4. Add MongoDB URI
5. Run `npm run dev`
